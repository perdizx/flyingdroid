Selamat datang di FlyingDroid Game!

Deskripsi Game:
FlyingDroid adalah permainan seru di mana Anda mengendalikan sebuah droid yang terbang melewati berbagai rintangan. Cobalah untuk mencapai skor tertinggi dengan menghindari rintangan dan mengumpulkan poin.

Cara Bermain:
- Tekan Enter untuk memulai permainan.
- Gunakan tombol Panah Atas atau Spasi untuk membuat droid terbang.
- Hindari rintangan dan kumpulkan poin untuk meningkatkan skor Anda.
- Permainan berakhir jika droid bertabrakan dengan rintangan.
